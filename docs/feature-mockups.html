safe undo

For destructive ops (compress, burn-in, overwrite), stash the original file in a temp cache for a window of time with a one-click undo. Nothing worse than a bad encode overwriting your only copy. Auto-purges after an hour or on quit.

CRT scanline mode

The crtEnabled flag we deleted could come back and actually do something: scanlines, phosphor glow, subtle flicker, vignette, optional green-tint phosphor. Toggle in settings. Pure CSS ::after overlay — zero perf cost. Peak terminal aesthetic.

rune casting

A playful "magic 8-ball" for indecisive users. Click cast and three runes fall — each maps to a suggested operation based on the file type (ᚦ = transform, ᚱ = reduce, ᛟ = extract). Entirely cosmetic framing on smart suggestions. Lean into the theme hard.

custom filter runes

Let power users author raw ffmpeg -vf filter chains and save them as runes. Live preview, syntax highlighting (we have it), and a small library of community presets. Turns galdr into a visual ffmpeg IDE — the killer feature for nerds.

AI auto-edit

Use ffmpeg's silencedetect + scene analysis to automatically find and cut dead air, jump-cut on scene changes, or extract highlights. One-click "make this 40-min lecture watchable." Pairs with whisper timestamps to cut on sentence boundaries. Local, no cloud.

An ASCII-art tree visualization of your output folders, styled as the world tree. Folders become branches, files become leaves, and processing status glows on each (● running, ✓ done, ✗ failed). A weirdly beautiful way to browse your library. Right-click a leaf to act on it.

ascii waveform

Render audio waveforms and spectrograms as terminal-style ASCII art using block characters (▁▂▃▄▅▆▇█). On-brand, lightweight, and genuinely useful for spotting silence/clipping. Click a region to select it for trimming or normalization. Weird but practical.
